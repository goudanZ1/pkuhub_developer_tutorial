#### 项目文件结构
注意看引用关系
```bash

.

├── sections
│   ├── _template.tpy  # 模板文件, 用于渲染单个章节
│   ├── 0.tpy          # Introducion 章节
│   ├── 1.tpy          # xj 的 vscode & git 章节
├── main.tpy           # 主文件, 用于打包渲染所有章节
├── global.tpy         # 全局变量
```
